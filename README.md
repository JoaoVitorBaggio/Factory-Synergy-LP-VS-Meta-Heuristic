# Seleção de Equipamentos de Fábrica
O trabalho consiste em um experimento que compara programação linear (LP) com meta heurística, nesse caso, busca tabu.

O problema escolhido para ser resolvido é da alocação de equipamentos com tabela de sinergias, que simula uma situação onde existem diversos equipamentos para serem ocupados, onde ele possuem diferentes custos, potências e sinergias entre si. Para ambas as soluções foi usado Python, sendo que a busca tabu foi completamente implementada, enquanto a LP foi modelada e resolvida usando Pulp.

O resumo dos resultados é que a busca tabu foi mais eficiente em encontrar resultados bons o suficiente, enquanto a LP foi mais rápida em achar a solução ótima.
A pasta do projeto conta com enunciado e logs dos resultados obtidos.
